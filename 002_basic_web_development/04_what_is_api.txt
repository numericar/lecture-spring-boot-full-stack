What is API ?
    API ย่อมาจาก Application Programming Interface

    ทำหน้าที่เป็นตัวกลางระหว่าง ลูกค้า และ บริการหลังบ้าน

    [client] <---> [api] <---> [service]

    โดยถ้าเปรียบเทียบกับร้านอาหาร จะเปรียบได้ดังนี้
        - คนมากิน = แอปพลิเคชันที่มาร้องขอบริการ/ข้อมูล
        - พ่อครัว = ระบบหลังบ้านต่าง ๆ 
        - เมนู = สเป็คของ api 
        - พนักงานเสิร์ฟ = API 
        - อาหาร = ข้อมูลที่ตอบกลับคำขอของผู้ใช้

    [client] <--- <internet> ---> [server] <---> [database]

    โดย API จะมีทั้งหมด 3 แบบ คือ 
        1. Private      ใช้ได้เฉพาะในองกรค์
        2. Partner      ใช้่ได้เฉพาะในองกรค์และคนที่เราอนุญาติ
        3. Public       ทุกคนสามารถใช้ได้
