Http/Https
    เป็นโปรโตคอลที่ใช้ในการส่งข้อมูลผ่านอินเทอร์เน็ต

    * HTTP = Hyper Text Transfer Protocol

    HTTPS เหมือนกับ HTTP แต่เพิ่มความปลอดภัยเข้ามา

Protocol
    เป็นกฏหรือข้อตกลงที่ใช้ในการสื่อสารระหว่างอุปกรณ์หรือโปรแกรมคอมพิวเตอร์ เพื่อให้เข้าใจและ
    สามารถแลกเปลี่ยนข้อมูลกันได้อย่างถูกต้อง
    เช่น 
        - HTTP/HTTPS    รับส่งข้อมูลบนเว็บไซต์
        - FTP           ส่งไฟล์
        - TCP/IP        การสื่อสารระดับพื้นฐานบนอินเทอร์เน็ต
